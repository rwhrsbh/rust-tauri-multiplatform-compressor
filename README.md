# Game Compressor

Кроссплатформенное десктопное приложение (Tauri v2 + Rust + React + Tailwind CSS) для **прозрачного сжатия игр** средствами файловой системы — игра занимает меньше места, но запускается как обычно, без распаковки.

## Как это работает

| ОС | ФС | Механизм |
|---|---|---|
| Windows | NTFS | WOF (Windows Overlay Filter, алгоритм **LZX**) через `FSCTL_SET_EXTERNAL_BACKING`; при недоступности WOF — откат на классическое NTFS-сжатие (`FSCTL_SET_COMPRESSION`, LZNT1) |
| macOS | APFS / HFS+ | decmpfs (`com.apple.decmpfs`) через `ditto --hfsCompression` с атомарной подменой файла |
| Linux | Btrfs | `FS_IOC_SETFLAGS` (+`FS_COMPR_FL`) + `BTRFS_IOC_DEFRAG_RANGE` со сжатием **zstd** |

FAT32/exFAT/ext4/XFS и прочие ФС без прозрачного сжатия определяются автоматически — приложение показывает предупреждение и блокирует действия.

### Proton / Wine игры на Linux

Windows-игры, установленные через Steam Proton (или Wine), лежат обычными файлами в файловой системе хоста (`steamapps/common`, `compatdata`). Если библиотека Steam находится на **Btrfs**, сжатие полностью работает. Приложение автоматически определяет Proton/Wine-игры и показывает подсказку.

### Безопасность

- Исполняемые файлы (`.exe`, `.dll`, `.dylib`, `.so`, `.sys`) **не сжимаются** — исключаются ложные срабатывания античитов и проблемы с запуском (актуально и для Proton).
- Сжатие прозрачное: данные не изменяются, меняется только способ хранения. Кнопка «Вернуть в исходное состояние» выполняет полную декомпрессию.

## Функции

- Проверка ФС диска (NTFS/APFS/Btrfs/ext4/FAT32/…) и свободного места
- Анализ папки: количество файлов, размер, прогноз сжатия по типам содержимого (текстуры, звук, архивы…)
- Многопоточное сжатие (rayon) с паузой / возобновлением / отменой
- Живой прогресс: %, скорость, освобождённые мегабайты, лог файлов
- Интерфейс на 4 языках: русский, английский, украинский, немецкий (переключение в настройках)
- Тёмная неоновая тема

## Разработка

Требования: Node.js 20+, Rust 1.77+, [системные зависимости Tauri](https://v2.tauri.app/start/prerequisites/).

```bash
npm install
npm run tauri dev     # запуск в dev-режиме
npm run tauri build   # сборка нативных пакетов
```

## Сборка под все платформы

GitHub Actions (`.github/workflows/build.yml`) собирает артефакты при каждом пуше:

- **Windows**: `.msi`, `.exe` (NSIS) — x64
- **macOS**: `.dmg` — Apple Silicon (arm64) и Intel (x64)
- **Linux**: `.deb`, `.rpm`, `.AppImage` — x64

Готовые пакеты доступны в артефактах воркфлоу (вкладка Actions).
